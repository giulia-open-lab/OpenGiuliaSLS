Folder Hierarchy
================

The repository is divided into multiple folders.
This structure is very clear, and aims to keep the code organized and easy to navigate.
The main folders are:

- .github: Contains the GitHub Actions workflows.
- .idea / .vscode: Contains the configuration files for the IDEs.
- data: Contains some auxiliary data files that are required by Giulia.
- docs: Contains the source files of this documentation.
- env: Contains the conda environment files.
- examples: Contains some examples of how to use Giulia.
- giulia: Contains the source code of Giulia.
- scripts: Contains some auxiliary scripts that are required by Giulia.
- shadowing: Contains some MatLAB scripts used for generating shadowing files for Giulia.
- templates: Contains some templates that can be used by the developers to know the format of source files.
- tests: Contains regression tests for Giulia.
